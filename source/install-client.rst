Install client
*************

..  tabs::

    ..  group-tab:: C#

        The client is available via `Nuget <https://www.nuget.org/packages?q=signature.api.client>`_. Download :code:`Direct`-client or :code:`Portal`-client based on your use case.


    ..  group-tab:: Java

        The client is available via Maven: |build-status|.

        Javadoc is available at `javadoc.io/doc/no.digipost <https://www.javadoc.io/doc/no.digipost.signature/signature-api-client-java/4.0>`_.
