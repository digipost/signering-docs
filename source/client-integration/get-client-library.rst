Get a client library
*************************

..  tabs::

    ..  group-tab:: C#

        The client is available via `Nuget <https://www.nuget.org/packages?q=signature.api.client>`_.

        The library has several packages with the prefix :code:`Digipost.Signature.Api.Client`. If you are using the portal cases, use :code:`Digipost.Signature.Api.Client.Portal`, and for direct cases, use :code:`Digipost.Signature.Api.Client.Direct`.
        If you want to download signed documents from the archive, use :code:`Digipost.Signature.Api.Client.Archive`.

        Source code is available on `GitHub (digipost/signature-api-client-dotnet) <https://github.com/digipost/signature-api-client-dotnet>`_.

    ..  group-tab:: Java

        The client is available via `Maven (no.digipost.signature:signature-api-client-java) <https://central.sonatype.com/artifact/no.digipost.signature/signature-api-client-java>`_.

        Javadoc is available at `javadoc.io <https://javadoc.io/doc/no.digipost.signature/signature-api-client-java>`_.

        Source code is available on `GitHub (digipost/signature-api-client-java) <https://github.com/digipost/signature-api-client-java>`_.
